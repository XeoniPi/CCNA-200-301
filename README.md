# CCNA-200-301
📘 Comprehensive CCNA 200-301 study resources and lab practice. Includes networking fundamentals, IP connectivity, security essentials, automation basics, and hands-on Packet Tracer labs. Designed for learners preparing for **Cisco’s CCNA 200-301 certification exam** with clear notes, configs, and examples.


---
## 🔹 What’s Inside
- Networking Fundamentals (OSI, TCP/IP, IPv4/IPv6)
- Switching & Routing basics
- IP Connectivity & Services
- Security Essentials
- Automation & Programmability
- Hands-on labs with Packet Tracer

## 🎯 Purpose
This repository is designed to help learners prepare for the CCNA exam with **clear notes, configs, and practical labs**.  
It’s simple, organized, and easy to follow for both beginners and advanced learners.
